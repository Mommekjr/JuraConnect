## ✨ Features

- ☕ Start brewing remotely
- 📡 Wi-Fi connectivity
- 🏠 Native Home Assistant integration
- 📊 Machine statistics
- 🔧 Read service information
- 🔄 OTA firmware updates *(planned)*
- 🖨️ 3D printable mounting system
- ❤️ Open Source

## 🚀 Project Goals

JuraConnect aims to make Jura coffee machines smarter while keeping the installation clean, reversible and easy to reproduce.

The project focuses on:

- Reliable communication with Jura machines
- Seamless Home Assistant integration
- Beautiful 3D-printed mounting solutions
- Open hardware and open software
- Easy installation for everyone
